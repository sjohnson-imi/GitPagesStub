AEO-Light 2

What is AEO-Light 2?

AEO-Light 2 is a new generation of optical sound extraction software developed by the University of South Carolina in close cooperation with Tommy Aschenbach. The project is made possible by the generous support of the National Endowment for the Humanities. The official project webpage contains additional information about he project’s history.

 

How can I use AEO-Light 2?

 

AEO-Light 2 supports MAC OS, Windows and Linux. Source films scans must contain a horizontal overscan to include the full width of the optical soundtrack and vertical overscan proving some information from the previous and following film frames. Source input can configured as individual DPX or TIF files or any video source that can be played by FFmpeg.

 

If I don’t know how to compile source code can I still use AEO-Light 2?

 

Absolutely! AEO-Light 2 was created to help all film preservationist, not just those who with coding skills. Follow this links for the most recent compiled code:  Mac OS, Windows. (You Linux users out there will want to build this application on your own).
